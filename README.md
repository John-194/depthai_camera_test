# Depthai camera test

This repo is for debugging Depthai "ping was missed" crash.

## Requirements

1. Jetson Orin NX 8GB
2. OAK-D S2 PoE
3. Camera bootloader v0.0.26
4. Depthai-python v2.22

## Usage

1. Run:
```bash
chmod +x build_docker.sh run.sh
./build_docker.sh
./run.sh
```

2. Wait up to 1 hour.
